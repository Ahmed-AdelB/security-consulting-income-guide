    GLASSDOOR SECURITY CONSULTING SALARY COMPILATION 2025¶

    Comprehensive Salary Research: Security Consultants, Contractors & Expert Networks¶

    Research Date: December 24, 2025 Data Sources: Glassdoor, expert network platforms, industry salary surveys Geographic Focus: United States (with location-specific breakdowns) Methodology: Web search compilation of public Glassdoor data, employee submissions, contractor reviews

    EXECUTIVE SUMMARY¶

    Key Salary Findings 2025:¶

    Security Consultant Roles:

    • Entry-Level Security Consultant: $96,609 - $128,813 (avg $62/hr)
    • Cyber Security Consultant: $117,619 - $153,476 (avg $74/hr)
    • Senior Security Consultant: $147,687 - $194,138 (avg $93/hr)
    • Principal Security Consultant: $154,305 - $255,722 (avg $123/hr)

    Penetration Testing Roles:

    • Penetration Tester: $116,043 - $153,206 (avg $74/hr)
    • Senior Penetration Tester: $147,687 - $187,964 (avg $90/hr)
    • Top Earners: Up to $263,580 (90th percentile)

    Contractor Rates:

    • Cybersecurity Contractor: $85,567 - $112,220 (avg $54/hr)
    • Independent Security Consultant: $88,360 - $117,813 (avg $57/hr)
    • IT Security Consultant: $129,497 - $162,500 (avg $78/hr)

    vCISO/Fractional CISO:

    • Hourly Rate: $150 - $500/hr
    • Monthly Retainer: $1,600 - $20,000/month
    • Project-Based: $10,000 - $100,000+ per project

    Expert Network Consultants:

    • GLG: $400 - $1,000+/hour (1:1 calls), $40-70 for surveys
    • AlphaSights: $250 - $400/hour (typical)
    • Guidepoint: $110 - $1,000/hour (varies by expertise level)
    • Third Bridge: $150 - $1,000/hour (varies by seniority)

    PART 4: EXPERT NETWORK CONSULTANT RATES¶

    4.1 GLG (Gerson Lehrman Group)¶

    Full-Time Consultant Salaries:

    • Average: $126,415/year
    • Pay Range: $94,812 (25th) - $170,286 (75th)
    • Annual Range: $101,193 - $216,083
    • Hourly: $57 - $92/hour (avg $66/hr base + $6/hr additional)
    • Data Source: 20 salaries (October 2025)

    Independent Expert/Consultant Rates:

    • 1:1 Consulting Calls: $400 - $1,000+/hour (often exceeds $1,000)
    • Business Surveys: $40 - $70 for 15 minutes (~$160-280/hour equivalent)
    • Payment Terms: Within 2 weeks after call completion
    • Consultant Rating: 3.7/5 (46 reviews)

    Pros:

    • Great income for side work
    • Prescreened clients
    • Clockwork payment
    • Flexible hours

    Cons:

    • No compensation for client no-shows
    • Projects "too few and far between"
    • Income too part-time to count on
    • Too many irrelevant project opportunities sent

    4.2 AlphaSights¶

    Full-Time Consultant Salaries:

    • Total Pay: $126,809/year (median)
    • Base Pay: $102,724/year
    • Additional Pay: $24,084/year (bonus, stock, commission)
    • Hourly: $50 - $81/hour (avg $59/hr base + $5/hr additional)
    • Salary Trajectory: $78,040 - $142,989/year

    Expert Network Consultant Rates:

    • Quoted Rates: $250/hour (typical)
    • Compensation Rating: 3.6/5

    Warning:

    • Some consultants report being "quoted $250" but receiving "less than half that amount"
    • No compensation unless able to bill clients
    • Calls postponed/cancelled after expert time investment
